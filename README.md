# tree-sitter-uon

[tree-sitter](https://tree-sitter.github.io/tree-sitter/) грамматика для **UON
(Универсальная Объектная Нотация)** — формата данных,类似 JSON, но с ключевыми
словами на русском языке.

## Возможности

- **Объекты** `{}` с парами ключ-значение
- **Массивы** `[]`
- **Строки** в двойных кавычках с экранированием
- **Числа** (целые и с плавающей точкой)
- **Логические значения:** `истина` / `ложь`
- **Null:** `пусто`
- **Даты:** `YYYY-MM-DD`
- **Длительности:** `5д`, `3ч30мин`, `10с`, `2г` и т.д.
- **Идентификаторы** (ключи без кавычек) с поддержкой Unicode
- **Комментарии:** `//` однострочные и `/* */` многострочные

## Пример

```uon
{
  имя: "Привет, мир!",
  возраст: 42,
  рост: 1.85,
  активен: истина,
  пустота: пусто,
  дата_рождения: 1990-05-15,
  время_в_пути: 2ч30мин,
  теги: ["учеба", "работа"],
  заметка: // комментарий
    "текст"
}
```

## Использование

### Установка

```bash
npm install tree-sitter-uon
```

### Использование с tree-sitter (Node.js)

```javascript
const Parser = require("tree-sitter");
const UON = require("tree-sitter-uon");

const parser = new Parser();
parser.setLanguage(UON);

const tree = parser.parse('{ ключ: "значение" }');
console.log(tree.rootNode.toString());
```

### Редакторы

Добавьте поддержку UON в ваш редактор через tree-sitter:

- **VS Code:** установите любой плагин для tree-sitter
- **Neovim:** через `nvim-treesitter`
- **Helix:** встроенная поддержка tree-sitter грамматик

## Структура грамматики

| Правило      | Описание                                   |
| ------------ | ------------------------------------------ |
| `document`   | Корневой узел, последовательность значений |
| `object`     | Объект `{ ... }`                           |
| `pair`       | Пара `ключ: значение`                      |
| `array`      | Массив `[ ... ]`                           |
| `string`     | Строка в `" "`                             |
| `number`     | Целое или число с плавающей точкой         |
| `boolean`    | `истина` или `ложь`                        |
| `null`       | `пусто`                                    |
| `date`       | Дата в формате `YYYY-MM-DD`                |
| `duration`   | Длительность (напр. `2ч30мин`, `5д`)       |
| `identifier` | Идентификатор (буквы, цифры, `_`)          |
| `comment`    | `//` или `/* */` комментарии               |

## Разработка

```bash
# Сгенерировать парсер
npx tree-sitter generate

# Запустить тесты
npx tree-sitter test

# Собрать WASM
npx tree-sitter build --wasm
```
